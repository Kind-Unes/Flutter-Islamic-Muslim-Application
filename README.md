## المميزات
1. السبحة
2. القرآن الكريم مع إمكانية اختيار الآيات
   - تلاوة الآية
   - تفسير الآية (4 تفاسير)
   - ترجمة الآية
   - مشاركة الآية
   - إضافة علامة مرجعية
3. نظام الورد اليومي
4. الأذكار والتذكيرات
5. الأدعية
6. فيديوهات تعليمية إسلامية
7. مكتبة الكتب الإسلامية
8. تذكير المهام (العادات الإسلامية)
9. اتجاه القبلة
10. مواقيت الصلاة
11. ملصقات إسلامية
12. متجر متكامل لبيع المصاحف
13. دورات تعليمية
